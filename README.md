# community_health_analysis
Health data analysis project for community health indicators

Statistical analysis project built in R using a synthetic community health dataset to explore relationships between demographic, behavioral, and physiological health indicators.

The project combines:

Exploratory Data Analysis (EDA)
Hypothesis Testing
Correlation Analysis
Linear Regression
Logistic Regression
Trend Analysis
Data Visualization

Tools & Technologies
R
tidyverse
ggplot2
dplyr
lubridate
R Markdown

Key Analyses
BMI trends across age groups
Cholesterol distribution by age
Smoking status vs diagnosis risk
Family history vs disease risk
BMI vs systolic blood pressure correlation
Screen time vs depression score
Logistic regression for diagnostic risk prediction
Monthly medical cost trend analysis

Key Findings
BMI positively correlated with systolic blood pressure
Family history significantly increased diagnosis risk
Smoking status was associated with diagnosis risk
Higher screen time related to higher depression scores
Cholesterol levels increased across older age groups
